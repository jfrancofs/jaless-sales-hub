# JALESS ONE v1.1 - Reglas MIC agregadas

## Cordones
- Se venden por metro.
- Códigos ejemplo: `COR4.50NTL`, `COR4.50VTN`.
- NTL = NBR/Nitrilo.
- VTN = Vitón.
- Ejemplos aceptados:
  - `Cordón de nitrilo / 4 mm 25`
  - `Cordón vitón / 3 mm 10`
  - `O'ring por metro 6 mm 30`

## Billas de acero
- Se venden por unidad.
- Códigos métricos: `BL-10`.
- Códigos en pulgadas: `BL-7/8"P`.
- Ejemplos aceptados:
  - `Billas / 10mm 30`
  - `Billas cromadas / 7/8" 10`

## Bonded Seal / Arandela con Jebe
- Sinónimos: Bonded Seal, Arandela con jebe, ARJ, SCR, BS.
- Ejemplos aceptados:
  - `Arandela con jebe / perno 8 20`
  - `Arandela con jebe SCR-212 5`
  - `ARJ-8 10`

## Nota
Después de copiar este paquete, se debe reimportar el catálogo inteligente para regenerar claves de búsqueda de Cordones, Billas y Bonded Seal.
