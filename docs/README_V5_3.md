# JALESS ONE v5.3 — Cliente 360°

## Incluye

- Ficha comercial completa por cliente.
- Datos de contacto y accesos directos a WhatsApp/correo.
- Resumen de cotizaciones, monto histórico y ticket promedio.
- Descuentos vigentes por categoría.
- Últimas cotizaciones y vendedor responsable.
- Productos más cotizados por frecuencia, cantidad y monto.
- Edición de datos, descuentos y estado desde la misma pantalla.

## Instalación

Copiar las carpetas `src` y `docs` dentro del proyecto y aceptar reemplazos.

No requiere SQL nuevo.

Abrir:

`http://localhost:3000/admin/clientes`
