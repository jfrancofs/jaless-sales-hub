# JALESS ONE v4.1

## Módulo: Administración de clientes PRO

Esta versión mejora la pantalla `/admin/clientes`.

Incluye:

- Crear cliente desde el panel administrativo.
- Editar razón social, RUC, dirección, ciudad, contacto, teléfono, correo y condición de pago.
- Validación de RUC con 11 dígitos numéricos.
- Desactivar cliente sin eliminarlo.
- Reactivar clientes inactivos.
- Mostrar/ocultar clientes inactivos.
- Resumen comercial por cliente:
  - número de cotizaciones,
  - monto total cotizado,
  - última cotización.
- Acceso directo a edición de descuentos del cliente.

No requiere SQL nuevo.
