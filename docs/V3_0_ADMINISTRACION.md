# JALESS ONE v3.0 - Panel administrativo

Incluye:

- `/admin`: resumen general y accesos rápidos.
- `/admin/clientes`: búsqueda y edición básica de clientes.
- `/admin/descuentos`: descuentos por categoría y botón de descuentos estándar Jaless.
- `/admin/productos`: búsqueda de productos y actualización de precios sin IGV.

## Recomendación

Esta versión sigue pensada como sistema interno. Cuando se implemente login de vendedores y roles, se deberán volver a activar políticas RLS con permisos por usuario.
