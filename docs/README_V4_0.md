# JALESS ONE v4.0

Versión de consolidación visual y experiencia de usuario.

## Incluye

- Dashboard administrativo más profesional.
- Métricas rápidas de cotizaciones.
- Acciones rápidas para cotizador, clientes, productos y descuentos.
- Resumen de últimas cotizaciones.
- No requiere SQL nuevo.

## Instalación

Copiar `src` y `docs` al proyecto y reiniciar `npm run dev`.
