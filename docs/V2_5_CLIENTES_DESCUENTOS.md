# JALESS ONE v2.5

## Incluye

- Botón **+ Nuevo cliente** en el cotizador.
- Registro inmediato de razón social, RUC, dirección, ciudad, contacto, teléfono, correo y condición de pago.
- Botón **Descuentos** para configurar descuentos por categoría del cliente seleccionado.
- Aplicación automática de descuentos en la cotización.

## SQL requerido

Ejecutar en Supabase:

```sql
database/013_clientes_descuentos_admin.sql
```

## Categorías de descuento

- O'Rings
- Abrazaderas
- Seguros
- Cordones
- Bonded Seal
- Pines de Expansión
- Billas de Acero
- Pasadores de Horquilla
